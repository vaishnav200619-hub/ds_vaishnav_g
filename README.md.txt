Trader behavious Insights - Web3 Trading

This project analyzes trader behaviour using Bitcoin Fear and Greed Index and Historical trading Data

Tools used
 - Python
 - Pandas
 - Seaborn
 - Google Colab

Files
 - notebook_1.ipynb -> main analysis (Google colab)
 - csv_files/ -> Input datasets
 - outputs/ -> Charts and visual outputs
 - ds_report.pdf -> Summary Report

Key Insights
1.Traders use higher leverage using greed periods, indicating increased confidence and risk taking.

2.Average losses are higher during fear periods,suggesting panic driven trading behaviour

3.Buy trades dominate Greed,while sell trades increase during fear conditionsKey Insights
